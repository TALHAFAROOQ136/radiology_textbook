# Clinical Radiology for Medical Students - Book Constitution

## Project Overview

**Book Title**: Clinical Radiology for Medical Students: From Basics to Clinical Practice

**Target Audience**: Medical students (pre-clinical and clinical years), healthcare professionals seeking radiology fundamentals

**Book Format**: Interactive AI-native textbook built with Docusaurus, deployed on GitHub Pages

**Core Technologies**: 
- Docusaurus (documentation framework)
- Spec-Kit Plus (AI-powered content generation)
- Claude Code (development automation)
- OpenAI Agents/ChatKit (RAG chatbot)
- FastAPI (backend API)
- Neon Serverless Postgres (database)
- Qdrant Cloud (vector database)
- Better-Auth (authentication)

---

## Book Structure

## Special Features

- **Image Galleries**: High-quality radiological images with annotations
- **Interactive Quizzes**: After each chapter for self-assessment
- **Case Simulations**: Virtual patient cases requiring imaging interpretation
- **Video Tutorials**: Demonstrating positioning techniques and interpretation methods
- **3D Anatomy Models**: Interactive 3D visualizations of anatomy on imaging

- **RAG Chatbot**: Context-aware assistant answering questions from book content

- **User Profiles**: Track educational level (medical student, resident, etc.)

- **Bookmarks and Notes**: Save important sections
